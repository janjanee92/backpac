# backpac-hw



## Built With

* jdk 11
* gradle 7.1.1
* Spring Boot 2.5.2
* Spring Data JPA 2.5.2
* QueryDsl 4.4.0
* MySQl 8.0.25
* Swagger ui 3.0.0

## Getting Started

```
./gradlew clean build
java -jar ./build/libs/backpac_hw-0.0.1-SNAPSHOT.jar
```
